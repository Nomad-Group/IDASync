Typedef Documentation

typedef tid_t enum_t

simple netnode that provides information about an enum.

name(), cmt() and rptcmt() are kept in the usual form. memqty is kept in altval(-1).

bitfields:

  • have altval(-5) bit0 set to 1.
  • altval('M') keeps ptrs to nodes by bitmasks

enums:

  • all members are kept in altval('E',member_value)-1
typedef uval_t bmask_t

unsigned value that describes a bitmask (also see enum_t).

a bit mask is 32/64 bits. if a mask is one-bit, then ptr to const_t is kept immediately in altval('M'). otherwise, masknode.altval('E') keeps ptrs to const_t by values (the same as with enums).

typedef uval_t const_t

members of enums are kept as netnodes.

name(), cmt() and rptcmt() are kept in the usual form. altval(-2) is enum_t.
