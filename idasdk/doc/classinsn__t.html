insn_t Class Reference
Instruction

Detailed Description

Structure to hold information about an instruction.

This structure is filled by the analysis step of IDP and used by the emulation and conversion to text steps. The kernel uses this structure too. All structure fields except cs, ip, ea, and op_t::n,op_t::flags of Operands are initialized to zero by the kernel. The rest should be filled by ana().
