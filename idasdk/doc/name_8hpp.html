name.hpp File Reference

Detailed Description

Functions that deal with names.

A non-tail address of the program may have a name. Tail addresses (i.e. the addresses in the middle of an instruction or data item) can not have names.

Give a name anyway.

This function tries to give the specified name the specified address and tries to modify the name with a postfix ("_00") if the name already exists. It will try 100 different postfixes and fail only if all such names are already in use. For example, if the specified name is "xxx", the function will try to give names "xxx","xxx_0","xxx_2",... "xxx_99".

Parameters
ealinear address
namenew name
  • NULL: return 0
  • "" : return 0
maxlenif specified, the name will be truncated to have the specified maximum length
Returns
success

idaman char ida_export_data XlatAsciiName[256]

Translation used to build meaningful names of ascii strings.

Ascii strings are given a 'meaningful' name based on the first characters of the string itself. This array is used to translate characters which can't appear in identifiers to valid ones. This array is initialized with information from ida.cfg.
