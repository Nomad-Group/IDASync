#define CH_MULTI   0x02

The chooser will allow multi-selection (only for GUI choosers).

If multi-selection is enabled, a multi-selection 'del' callback will be called with:

  • uint32 = START_SEL before calling the first selected item
  • uint32 = 1..n for each selected item
  • uint32 = END_SEL after calling the last selected item

If the callback returns 'false', further processing of selected items is cancelled.

#define CH_MULTI_EDIT   0x04

The 'edit' callback will be called for all selected items using the START_SEL/END_SEL protocol.

This bit implies CH_MULTI (valid only for gui)

#define CH_NOBTNS   0x08
