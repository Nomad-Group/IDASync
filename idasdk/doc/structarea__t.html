area_t Struct Reference

Detailed Description

Base class for an area.

This class is used as a base class for a class with real information - see segment.hpp for example. The end address points beyond the area.

Inherited by func_t, hidden_area_t, memory_info_t, regvar_t, segment_t, and segreg_area_t.
