Display a window with a list (1-column) and allow the user to select an item.

Parameters
flagsGeneric chooser flags
x0,y0,x1,y1screen position
  • x0=-1 for autoposition
  • set x0=-2 if you need to use the chooser before opening the database
objobject to show
widthmax width of lines
sizernumber of items
getldescription of n-th item (1..n)
  • 0-th item if header line
  • 'buf' cannot be larger than MAXSTR
titlemenu title (includes ptr to help). may have chooser title prefixes (see above)
iconnumber of the default icon to display
defltitem number to focus (1-based)
  • -1 if a non-modal chooser was already open, do not change selection
  • otherwise focus on the first item
delcallback for "Delete" (may be NULL)
  • supports multi-selection scenario too
  • returns: 1-ok, 0-failed
inscallback for "New" (may be NULL)
updatecallback for "Update"(may be NULL)
  • update the whole list
  • returns the new location of item 'n'
editcallback for "Edit" (may be NULL)
entercallback for non-modal "Enter" (may be NULL)
destroycallback to call when the window is closed (may be NULL)
popup_namesdefault: insert, delete, edit, refresh
get_iconcallback for get_icon (may be NULL)
Return values
-1the chooser was already open and is now active (only for non-modal choosers)
0the chooser was created but the user refused to choose anything
elsenumber of them selected item
