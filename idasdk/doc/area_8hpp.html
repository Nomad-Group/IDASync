Detailed Description

Contains the definition of areacb_t.

This is a base class used by many parts of IDA. It is a collection of address ranges in the program. It conceptually consists of separate area_t instances.

An area is a non-empty contiguous range of addresses (specified by its start and end addresses, the end address is excluded from the range) with some characteristics. For example, the ensemble of program segments is represented by an "areacb_t" called "segs".

Areas are stored in the Btree part of the IDA database. To learn more about Btrees (Balanced Trees): http://www.bluerwhite.org/btree/
